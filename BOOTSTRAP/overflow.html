The .overflow-auto automatically adjust the contents in the container class. It add vertically scrolling to the element.

The .overflow-hidden automatically adjust the contents in the container class. It add vertically scrolling to the element.

The .overflow-show automatically adjust the contents in the container class. It add vertically scrolling to the element.

The .overflow-scroll automatically adjust the contents in the container class. It add vertically scrolling to the element.
